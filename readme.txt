== Description ==
This plugin is an addon for WP Church Center. It adds a 'Tithe.ly' Giving card which allows churches to have giving initiated through their 'center'. You'll need to have an account from https://get.tithe.ly/ to use this card). (You'll need at least v1.2.7 of WP Church Center to use this Addon).

== Installation ==
1). Download the plugin

2). Install the plugin by either:
extracting the .zip file and uploading the entire wpcc-tithely folder to the /wp-content/plugins/ directory of our web server.
go to Plugins > Add New in the WordPress Admin and upload the .zip file.
(You'll need at least v1.2.7 of WP Church Center to use this Addon).

3). Activate the Plugin. You should now be able to select 'Tithe.ly Giving' as a card type.

NOTE: Your server will require an SSL certificate in order for payments to be processed through the Tithe.ly system. Talk to your web host for further details.
